# Enabling ctron_plugin module in Clusterio scenario loader

Clusterio generates the Factorio `control.lua` loader for each save/scenario.  To load this module, add:

```lua
event_handler.add_lib(require("modules/ctron_plugin/control"))
require("modules/ctron_plugin/globals")
```

near the other `event_handler.add_lib(...)` lines.

> Note: The loader file is auto-generated by Clusterio; changes may need to be applied through the normal Clusterio module/plugin deployment flow for your setup.
